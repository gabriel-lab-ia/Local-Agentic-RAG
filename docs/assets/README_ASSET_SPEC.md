# README Banner Asset Specification

> Status: implemented in [`local-agentic-rag-banner.svg`](local-agentic-rag-banner.svg).

Create a professional banner for Local Agentic RAG.

## Concept

- Dark industrial AI infrastructure background.
- Clean technical composition, not futuristic exaggeration.
- Central cognitive core labeled generically as local Qwen inference.
- JSON schema panel as the first visual signal.
- Linux terminal panel as the runtime substrate.
- Vector nodes flowing into a ChromaDB-like semantic memory block.
- RAG flow arrows from corpus to embeddings to retrieval to cited answer.
- Security boundary layer around tools and filesystem access.
- No vendor logos unless explicit permission exists.

## Style

- Wide banner, recommended 1600x520.
- Dark charcoal background with restrained blue, green, and amber accents.
- Crisp lines, subtle grid, high contrast text.
- Avoid stock-photo people, mascots, or glowing fantasy effects.
- Use labels: JSON Contract, Linux Runtime, Qwen Local Inference, EmbeddingGemma, ChromaDB, Read-only Tools, Security Boundary.
