# Code of Conduct

This project follows a simple standard: be respectful, precise, and constructive.

## Expected Behavior

- Discuss ideas and code with professionalism.
- Assume good intent while still reviewing carefully.
- Keep security reports responsible and sanitized.
- Respect privacy and local-first boundaries.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing secrets, private logs, or sensitive local data.
- Encouraging unsafe execution, destructive commands, or credential exposure.
- Misrepresenting experimental features as production-ready.

Maintainers may remove content or contributions that violate these expectations.
